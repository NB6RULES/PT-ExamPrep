🔧 Module 1

Kinematics of Machine Tools

Speed & feed selection, gear box design, ray diagrams, structural diagrams, and speed mechanisms in machine tools. (Prof. Pramod — PRK)

🔍 Introduction

Kinematics of machine tools deals with the study of motion (without considering forces) in machines used for manufacturing operations. It focuses on how cutting tools and workpieces move relative to each other to produce the desired shape, size, and surface finish.

Key Concept Material removal occurs due to relative motion between the cutting tool and the workpiece.

Examples of Relative Motion:

  • In a lathe — the workpiece rotates and the tool feeds linearly.
  • In a milling machine — the cutter rotates and the workpiece feeds.
  • In a drilling machine — the drill rotates and feeds into the workpiece.

⚙️ Types of Motions in Machine Tools

1. Primary (Cutting) Motion

  • Responsible for the cutting action.
  • Usually rotary (lathe, milling, drilling) or linear (shaping, planing).
  • Provides the cutting speed.

2. Feed Motion

  • Moves the tool/work gradually for continuous cutting.
  • Slower than primary motion.
  • Determines the depth of cut and chip thickness.

3. Auxiliary Motions

  • Tool positioning, clamping, return stroke (in shaping)

🎯 Speed & Feed Selection

Major Factors Affecting Speed Selection

  • Work material: Harder materials require lower cutting speeds.
  • Tool material: HSS, carbide, ceramic tools allow different speeds.
  • Depth of cut: Heavier cuts require reduced speed.
  • Feed rate: Higher feeds reduce permissible speed.
  • Type of operation: Rough vs. finish machining.
  • Coolant use: Coolant allows higher speeds.
  • Machine rigidity: Affects maximum permissible speed.

Cutting Speed Formula

V = (π × D × N) / 1000 [m/min] where D = diameter (mm), N = speed (rpm)

Major Factors Affecting Feed Selection

  • Surface finish requirement (finer finish → lower feed)
  • Workpiece material strength
  • Tool geometry (nose radius, rake angle)
  • Type of operation (roughing vs. finishing)
  • Rigidity of work and machine

📐 Progression Ratio (Step Ratio)

When spindle speeds are arranged in Geometric Progression (GP), the ratio between two adjacent speeds is the progression ratio (φ).

φ = N₂/N₁ = N₃/N₂ = ... = (Nmax/Nmin)^(1/(n−1)) where: φ = progression ratio n = number of speed steps Nmin = minimum speed (rpm) Nmax = maximum speed (rpm) Permissible deviation = ±10%

Why Geometric Progression?

  • Speed loss is minimum (Speed loss = Desired optimum speed − Available speed)
  • Minimum number of gears required.
  • More even range of spindle speeds at each step.
  • Very compact layout.
  • Productivity (surface area removed/unit time) is constant across the speed range.
  • Spindle speeds can be easily selected from preferred numbers.

🔢 Preferred Numbers

Preferred numbers are conventionally rounded-off values derived from geometric series. Five basic series:

SeriesStep Ratio (φ)Use
R 51.58Coarse series
R 101.26General use
R 201.12Machine tool gear boxes
R 401.06Fine gear boxes
R 801.03Very fine series
Exam Tip R20 and R40 series are most commonly used in the design of multi-speed gear boxes.

Calculation of Non-Standard Speeds

N₁ = Nmin N₂ = Nmin × φ¹ N₃ = Nmin × φ² ... Nₙ = Nmin × φ^(n-1) Number of speeds to skip = ln(φ_calculated) / ln(φ_standard)

🏗️ Structural Diagram

A structural diagram is a kinematic layout that shows the arrangement of gears in a gear box — shafts, gear pairs, and the number of speed steps at each stage.

Structural Formula for n-speed Gear Box

General form: P₁(X₁) × P₂(X₂) × P₃(X₃)

  • P = number of gear pairs in that stage
  • X = characteristic (number of speeds before that stage)

Possible Arrangements

No. of SpeedsStructural Formulas
6 speeds3(1)×2(3)  |  2(1)×3(2)
8 speeds2(1)×2(2)×2(4)  |  4(1)×2(4)
9 speeds3(1)×3(3)
12 speeds3(1)×2(3)×2(6)  |  2(1)×3(2)×2(6)  |  2(1)×2(2)×3(4)

Rules for Gear Box Design

  • Transmission ratio: 1/4 ≤ i ≤ 2
  • Speed ratio at any stage must not exceed 8: φ^P ≤ 8
  • In all stages except the first: X ≥ P_prev
  • Sum of teeth of mating gears must be same for same module in a sliding set.
  • Minimum number of teeth on smallest gear ≥ 17.

📊 Ray Diagram

A ray diagram (speed chart) is a graphical representation of the drive arrangement that determines the specific values of all transmission ratios and speeds of all shafts in the drive.

Difference: Structural Diagram vs Ray Diagram

Structural DiagramRay Diagram
Kinematic layout showing shaft and gear arrangementGraphical representation of drive arrangement showing speed values
Shows how many speeds at each stageShows exact speed values on each shaft
Does not show actual speed valuesSpeed scale is logarithmic (equal intervals for GP)
Used for initial design layoutUsed to determine and verify transmission ratios

How to Draw a Ray Diagram

  1. Draw vertical lines for each shaft (input to output).
  2. Mark speed values on a logarithmic scale (horizontal lines).
  3. Input shaft has one speed; each successive shaft shows speed steps.
  4. Connect speeds with lines representing gear pairs (rays).
  5. Slope of ray = transmission ratio of that gear pair.
  6. Diverging rays = speed increase; converging rays = speed decrease.

⚙️ Gear Box Design

Requirements of a Speed Gear Box

  • Must provide the designed series of spindle speeds.
  • Must transmit the required amount of power to the spindle.
  • Must provide smooth, silent operation of the transmission.
  • Simple construction and easily accessible for maintenance.

Methods of Changing Speed

  • Sliding mesh gear box — spur gears slid along splines to mesh with different gears.
  • Constant mesh gear box — helical gears always in mesh; clutches select speed.
  • Synchromesh gear box — synchronizer rings allow smooth engagement.
  • Planetary gear box — compact, high torque applications.
Key Rule In a sliding mesh gear box, the sum of teeth of mating gears must be equal in the same stage (same module) to allow sliding. Helical gears are used in constant mesh type for quieter, smoother operation.

🔩 Speed & Feed Mechanisms in Machines

Speed Mechanism in Lathe

In a lathe, the speed mechanism transmits power from the motor to the spindle through a series of gear stages. Common arrangement:

  • Back gear arrangement: Provides extra low-speed range for heavy cuts (threading, boring).
  • Motor → V-belt → Countershaft → Headstock gear box → Spindle.
  • Spindle speeds selected by shifting gear clusters on the headstock.
  • Norton gear box (tumbler gear) is used for feed mechanism in lathe to select different thread pitches and feeds.

Speed Mechanism in Milling Machine

  • Motor drives the spindle through a multi-speed gear box (usually 18 or more speeds).
  • Feed is provided by a separate feed gear box (independent from speed).
  • Both horizontal and vertical spindles can be driven in a universal milling machine.

Feed & Speed Mechanism in Drilling Machine

  • Spindle speed: achieved through step pulleys or gear box.
  • Feed: by hand or through automatic feed mechanism — lead screw or rack and pinion.
  • Sensitive drill press uses hand feed; upright drill press uses gear-driven automatic feed.

✏️ Solved Problems

Problem 1 — Find progression ratio and spindle speeds for a 12-speed gear box (100 to 355 rpm)

Given: n = 12, Nmin = 100 rpm, Nmax = 355 rpm

1
Progression ratio:
φ = (Nmax/Nmin)^(1/(n-1)) = (355/100)^(1/11) = (3.55)^(0.0909) = 1.122 ≈ 1.12 (R20 series)
2
Spindle speeds from R20 series (φ = 1.12):
100, 112, 125, 140, 160, 180, 200, 224, 250, 280, 315, 355 rpm
3
Verification: Since φ calculated = 1.12 matches R20 series standard step ratio, all speeds are standard preferred numbers. ✓
Problem 2 — Design a 6-speed gearbox (2×3), common ratio = 1.2, input shaft speed = 300 rpm, motor speed = 1500 rpm, Power = 16 HP

Given: 6-speed, φ = 1.2, Ninput = 300 rpm, Nmotor = 1500 rpm, P = 16 HP

1
Structural formula: 2(1) × 3(2)
Stage 1: 2 gear pairs, Stage 2: 3 gear pairs
2
6 output speeds:
N₁ = Nmin
N₂ = N₁ × 1.2
N₃ = N₁ × 1.2² = N₁ × 1.44
N₄ = N₁ × 1.2³ = N₁ × 1.728
N₅ = N₁ × 1.2⁴ = N₁ × 2.0736
N₆ = N₁ × 1.2⁵ = N₁ × 2.4883
3
Transmission ratios for Stage 1 (2 gear pairs):
i₁ = 1 (direct), i₂ = 1/φ³ (for 2(1) arrangement, characteristic = 1)
The two rays diverge/converge by φ step.
4
Ray diagram: Draw shaft I → II with 2 rays (Stage 1), then shaft II → III with 3 rays each (Stage 2), giving 2 × 3 = 6 speeds at output shaft.
Problem 3 — 6(3×2) speed gear box, 600–1600 rpm, 10 HP — Ray diagram & shaft sizes

Given: 6-speed, 3×2, Nmin = 600 rpm, Nmax = 1600 rpm, P = 10 HP = 7.46 kW

1
φ = (1600/600)^(1/5) = (2.667)^0.2 = 1.217
2
Speeds: 600, 730, 888, 1080, 1315, 1600 rpm
3
Shaft torque (for shaft sizing):
T = P/ω = 7460 / (2π × N/60)
At minimum speed (600 rpm): T_max = 7460 / (2π × 600/60) = 7460/62.83 = 118.7 N·m
4
Shaft diameter: d = (16T / π × τ_allow)^(1/3)
Assuming τ_allow = 55 MPa for steel:
d = (16 × 118.7 / (π × 55×10⁶))^(1/3) = (1.1×10⁻⁵)^(1/3) ≈ 22 mm (use standard 25 mm)
