🔧 Module 1

Kinematics of Machine Tools

Speed & feed selection, gear box design, ray diagrams, structural diagrams, and speed mechanisms in machine tools. (Prof. Pramod — PRK)

🔍 Introduction

Kinematics of machine tools deals with the study of motion (without considering forces) in machines used for manufacturing operations. It focuses on how cutting tools and workpieces move relative to each other to produce the desired shape, size, and surface finish.

Key Concept Material removal occurs due to relative motion between the cutting tool and the workpiece.

Examples of Relative Motion:

  • In a lathe — the workpiece rotates and the tool feeds linearly.
  • In a milling machine — the cutter rotates and the workpiece feeds.
  • In a drilling machine — the drill rotates and feeds into the workpiece.

⚙️ Types of Motions in Machine Tools

1. Primary (Cutting) Motion

  • Responsible for the cutting action.
  • Usually rotary (lathe, milling, drilling) or linear (shaping, planing).
  • Provides the cutting speed.

2. Feed Motion

  • Moves the tool/work gradually for continuous cutting.
  • Slower than primary motion.
  • Determines the depth of cut and chip thickness.

3. Auxiliary Motions

  • Tool positioning, clamping, return stroke (in shaping)

🎯 Speed & Feed Selection

Major Factors Affecting Speed Selection

  • Work material: Harder materials require lower cutting speeds.
  • Tool material: HSS, carbide, ceramic tools allow different speeds.
  • Depth of cut: Heavier cuts require reduced speed.
  • Feed rate: Higher feeds reduce permissible speed.
  • Type of operation: Rough vs. finish machining.
  • Coolant use: Coolant allows higher speeds.
  • Machine rigidity: Affects maximum permissible speed.

Cutting Speed Formula

V = (π × D × N) / 1000 [m/min] where D = diameter (mm), N = speed (rpm)

Major Factors Affecting Feed Selection

  • Surface finish requirement (finer finish → lower feed)
  • Workpiece material strength
  • Tool geometry (nose radius, rake angle)
  • Type of operation (roughing vs. finishing)
  • Rigidity of work and machine

📐 Progression Ratio (Step Ratio)

When spindle speeds are arranged in Geometric Progression (GP), the ratio between two adjacent speeds is the progression ratio (φ).

φ = N₂/N₁ = N₃/N₂ = ... = (Nmax/Nmin)^(1/(n−1)) where: φ = progression ratio n = number of speed steps Nmin = minimum speed (rpm) Nmax = maximum speed (rpm) Permissible deviation = ±10%

Why Geometric Progression?

  • Speed loss is minimum (Speed loss = Desired optimum speed − Available speed)
  • Minimum number of gears required.
  • More even range of spindle speeds at each step.
  • Very compact layout.
  • Productivity (surface area removed/unit time) is constant across the speed range.
  • Spindle speeds can be easily selected from preferred numbers.

🔢 Preferred Numbers

Preferred numbers are conventionally rounded-off values derived from geometric series. Five basic series:

SeriesStep Ratio (φ)Use
R 51.58Coarse series
R 101.26General use
R 201.12Machine tool gear boxes
R 401.06Fine gear boxes
R 801.03Very fine series
Exam Tip R20 and R40 series are most commonly used in the design of multi-speed gear boxes.

Calculation of Non-Standard Speeds

N₁ = Nmin N₂ = Nmin × φ¹ N₃ = Nmin × φ² ... Nₙ = Nmin × φ^(n-1) Number of speeds to skip = ln(φ_calculated) / ln(φ_standard)

🏗️ Structural Diagram

A structural diagram is a kinematic layout that shows the arrangement of gears in a gear box — shafts, gear pairs, and the number of speed steps at each stage.

Structural Formula for n-speed Gear Box

General form: P₁(X₁) × P₂(X₂) × P₃(X₃)

  • P = number of gear pairs in that stage
  • X = characteristic (number of speeds before that stage)

Possible Arrangements

No. of SpeedsStructural Formulas
6 speeds3(1)×2(3)  |  2(1)×3(2)
8 speeds2(1)×2(2)×2(4)  |  4(1)×2(4)
9 speeds3(1)×3(3)
12 speeds3(1)×2(3)×2(6)  |  2(1)×3(2)×2(6)  |  2(1)×2(2)×3(4)

Rules for Gear Box Design

  • Transmission ratio: 1/4 ≤ i ≤ 2
  • Speed ratio at any stage must not exceed 8: φ^P ≤ 8
  • In all stages except the first: X ≥ P_prev
  • Sum of teeth of mating gears must be same for same module in a sliding set.
  • Minimum number of teeth on smallest gear ≥ 17.

📊 Ray Diagram

A ray diagram (speed chart) is a graphical representation of the drive arrangement that determines the specific values of all transmission ratios and speeds of all shafts in the drive.

Difference: Structural Diagram vs Ray Diagram

Structural DiagramRay Diagram
Kinematic layout showing shaft and gear arrangementGraphical representation of drive arrangement showing speed values
Shows how many speeds at each stageShows exact speed values on each shaft
Does not show actual speed valuesSpeed scale is logarithmic (equal intervals for GP)
Used for initial design layoutUsed to determine and verify transmission ratios

How to Draw a Ray Diagram

  1. Draw vertical lines for each shaft (input to output).
  2. Mark speed values on a logarithmic scale (horizontal lines).
  3. Input shaft has one speed; each successive shaft shows speed steps.
  4. Connect speeds with lines representing gear pairs (rays).
  5. Slope of ray = transmission ratio of that gear pair.
  6. Diverging rays = speed increase; converging rays = speed decrease.

⚙️ Gear Box Design

Requirements of a Speed Gear Box

  • Must provide the designed series of spindle speeds.
  • Must transmit the required amount of power to the spindle.
  • Must provide smooth, silent operation of the transmission.
  • Simple construction and easily accessible for maintenance.

Methods of Changing Speed

  • Sliding mesh gear box — spur gears slid along splines to mesh with different gears.
  • Constant mesh gear box — helical gears always in mesh; clutches select speed.
  • Synchromesh gear box — synchronizer rings allow smooth engagement.
  • Planetary gear box — compact, high torque applications.
Key Rule In a sliding mesh gear box, the sum of teeth of mating gears must be equal in the same stage (same module) to allow sliding. Helical gears are used in constant mesh type for quieter, smoother operation.

🔩 Speed & Feed Mechanisms in Machines

Speed Mechanism in Lathe

In a lathe, the speed mechanism transmits power from the motor to the spindle through a series of gear stages. Common arrangement:

  • Back gear arrangement: Provides extra low-speed range for heavy cuts (threading, boring).
  • Motor → V-belt → Countershaft → Headstock gear box → Spindle.
  • Spindle speeds selected by shifting gear clusters on the headstock.
  • Norton gear box (tumbler gear) is used for feed mechanism in lathe to select different thread pitches and feeds.

Speed Mechanism in Milling Machine

  • Motor drives the spindle through a multi-speed gear box (usually 18 or more speeds).
  • Feed is provided by a separate feed gear box (independent from speed).
  • Both horizontal and vertical spindles can be driven in a universal milling machine.

Feed & Speed Mechanism in Drilling Machine

  • Spindle speed: achieved through step pulleys or gear box.
  • Feed: by hand or through automatic feed mechanism — lead screw or rack and pinion.
  • Sensitive drill press uses hand feed; upright drill press uses gear-driven automatic feed.

Apr 2024 · Part A (I.a)Explain briefly the term progression ratio.
3 marksCO1L2
View Answer

Progression Ratio (φ) is the constant ratio between two consecutive spindle speeds when the speeds are arranged in Geometric Progression (GP).

φ = N₂/N₁ = N₃/N₂ = N₄/N₃ = ... = constant φ = (N_max / N_min)^(1/(n−1)) where: N_max = maximum spindle speed (rpm) N_min = minimum spindle speed (rpm) n = total number of speed steps

Why GP is preferred:

  • Speed loss (= desired speed − available speed) is minimum and uniform across all speed steps.
  • Requires minimum number of gears for a given speed range.
  • Productivity (metal removal rate) remains nearly constant across the entire speed range.
  • Spindle speeds can be directly selected from standard preferred number series (R5, R10, R20, R40).

Permissible deviation: ±10% from the calculated step ratio is acceptable.

A speed gear box must satisfy the following requirements:

  • Speed range: Must provide the designed series of spindle speeds (all required speed steps).
  • Power transmission: Must transmit the required amount of power to the spindle without failure.
  • Smooth operation: Must provide smooth and silent operation of the transmission mechanism at all speeds.
  • Simple construction: The gear box should have a simple, compact design to reduce manufacturing cost.
  • Maintenance: The mechanism should be easily accessible so that preventive maintenance can be carried out conveniently.
  • Efficiency: Minimum power loss due to friction in gear meshes and bearings.
  • Rigidity: Sufficient rigidity to resist deflection under cutting loads.

Speed Mechanism in a Lathe

The speed mechanism in a lathe transmits power from the electric motor to the spindle at different speeds to suit different workpiece materials, diameters, and operations.

Power Flow Path

Electric Motor → V-belt → Countershaft (constant speed) → Headstock gear box → Spindle

1. Motor and Belt Drive

  • An electric motor (usually 3-phase induction motor) provides the primary power.
  • Power is transmitted through a V-belt drive to the countershaft, which rotates at a constant speed.
  • The V-belt provides slip protection and dampens vibrations.

2. Countershaft (Intermediate Shaft)

  • Rotates at constant speed (driven by motor via V-belt).
  • Carries a step pulley or a single pulley depending on lathe type.
  • In older lathes: a cone/step pulley on countershaft meshes with a cone pulley on headstock to give 3–4 speeds.

3. Headstock Gear Box

  • Contains multiple sliding gear clusters mounted on splined shafts.
  • By shifting gear clusters using levers, different gear combinations are engaged.
  • Provides a wide range of spindle speeds (typically 8 to 24 steps in geometric progression).
  • All shafts are supported in anti-friction bearings.

4. Back Gear Arrangement

  • Provides extra-low speed range for heavy-duty operations (threading, boring large diameters).
  • Consists of two back gears on a shaft parallel to the spindle.
  • When back gear is engaged: drive passes through back gears before reaching spindle → much lower speed and higher torque.
  • When back gear is disengaged: direct drive from cone pulley to spindle → higher speeds for light cuts.
  • Doubles the number of available speeds (e.g., 8 direct + 8 back-gear = 16 speeds).

5. Feed Mechanism — Norton (Tumbler) Gear Box

  • The spindle drives the feed shaft through a tumbler gear (reversing mechanism) and Norton gear box.
  • The Norton gear box is a quick-change gear box containing a cone of gears.
  • A sliding key engages one gear in the cone at a time, giving different feed rates and thread pitches.
  • The selected gear drives the lead screw (for threading) or feed rod (for turning/facing feeds).

6. Spindle

  • The spindle is the final output shaft — it carries and rotates the workpiece (via chuck, face plate, or between centres).
  • Made of alloy steel; supported in precision taper roller or angular contact bearings.
  • Has a Morse taper bore at the front for mounting centres and collets.
Key Point for Exam In an all-geared headstock lathe, the V-belt drive to the headstock is replaced by a geared drive directly from the motor, giving positive, slip-free transmission and a wider speed range without belt maintenance issues.

3 marksCO1L2
View Answer

A ray diagram (also called a speed chart or kinematic diagram) is a graphical representation of the drive arrangement of a machine tool gear box that shows the specific values of all transmission ratios and speeds at every shaft stage.

Construction:

  • Vertical lines represent the shafts (input shaft on left, output shaft on right).
  • Horizontal lines represent speed values on a logarithmic scale — equal spacing because speeds are in geometric progression.
  • Slanted lines (rays) connect speed points on adjacent shafts, representing individual gear pairs.
  • The slope of each ray indicates the transmission ratio of that gear pair.
  • A ray slanting upward = speed increase (step-up); slanting downward = speed decrease (step-down); horizontal = 1:1 ratio.

Purpose: To graphically verify the structural formula, find exact transmission ratios, and confirm that all required speeds are achieved within permissible limits.

Sep 2024 · Part A (I.b)Describe stepped and stepless regulation of speeds.
3 marksCO1L1
View Answer

Stepped Speed Regulation:

  • Speed is varied in discrete steps — only a finite number of fixed speeds are available.
  • Achieved using gear boxes (sliding mesh, constant mesh), step pulleys, or cone pulleys.
  • Speeds are arranged in geometric progression using a standard step ratio (φ).
  • Advantages: high efficiency, positive (slip-free) drive, low cost, simple construction.
  • Disadvantage: exact desired cutting speed may not be available; only the nearest step can be selected.
  • Applications: most conventional machine tools — lathes, milling machines, drilling machines.

Stepless Speed Regulation:

  • Speed can be varied continuously and infinitely within a range — any desired speed is achievable.
  • Achieved using: Variable Speed Drives (VSD), PIV (Positive Infinitely Variable) drives, hydraulic drives, DC/AC variable frequency motors, CVT (Continuously Variable Transmission).
  • Advantages: exact optimum cutting speed always available; smooth transition; higher productivity.
  • Disadvantages: higher cost, more complex mechanism, slightly lower efficiency.
  • Applications: CNC machines, heavy-duty machines, precision grinding, thread milling.

Speed and Feed Mechanisms in a Milling Machine

A milling machine has two independent mechanisms: (1) the speed mechanism which drives the spindle (cutter), and (2) the feed mechanism which drives the workpiece table.

1. Speed Mechanism

Power flow:

Electric Motor → V-belt/coupling → Speed Gear Box → Spindle
  • The speed gear box provides 8 to 18 spindle speeds in geometric progression.
  • Speeds are selected by shifting gear clusters using levers or handles on the gear box.
  • The spindle carries the arbor and milling cutter (in horizontal milling) or collet/tool (in vertical milling).
  • Spindle is supported in precision taper roller bearings to handle radial and axial cutting forces.
  • In a universal milling machine, the spindle can also be tilted at an angle.
  • Overarm: Supports the far end of the arbor in horizontal milling machines to prevent deflection.

2. Feed Mechanism

Power flow:

Spindle (or separate feed motor) → Feed Gear Box → Bevel gear set → Feed clutches → Table drives
  • The feed gear box provides multiple feed rates for the three table axes: longitudinal, cross (transverse), and vertical.
  • Only one feed direction is engaged at a time using feed clutches.
  • Feed rates are selected independently of spindle speed — they are controlled by separate levers on the feed gear box.
  • Feeds are expressed in mm/min (or mm/tooth for precision work).
  • The table is driven by lead screws (horizontal axes) and elevating screw (vertical axis).

3. Rapid Traverse

  • A separate rapid traverse motor (or clutch bypass) drives the table at high speed for positioning, bypassing the feed gear box.
  • This saves time during non-cutting movements.

4. Difference between Speed and Feed Drives

Speed Drive (Spindle)Feed Drive (Table)
Drives the milling cutterDrives the workpiece table
Higher speed range (rpm)Lower rate (mm/min)
Controlled by speed gear box leversControlled by feed gear box levers
Always running during machiningOne axis active at a time
